# 관광일본어과 AI 융합 수업 가이드

## 프로젝트 개요

관광일본어과 수업에서 Gemini Canvas로 일본 여행 회화 퀴즈를 만드는 활동을 안내하는 **단일 HTML 수업 가이드**입니다. 실제 산출물은 `index.html` 하나이며, 이 저장소 자체가 Gemini 연동 앱이나 회화 퀴즈 앱은 아닙니다.

페이지는 교사 시연용·학생 실습용 프롬프트와 수업 절차, 체험 체크리스트, 서비스 접속이 어려울 때의 Plan B를 보여 줍니다. 학생은 안내된 프롬프트를 별도의 Gemini Canvas에 직접 붙여 넣어 외부에서 결과물을 만드는 흐름을 전제로 합니다.

## 주요 기능

- 수업개요, 교사시연, 학생실습, 비상대응의 네 화면을 JavaScript로 탭 전환
- 교사·학생 프롬프트와 짧은 피드백 문구를 Clipboard API로 복사
- 공항 입국심사, 호텔 체크인, 식당 주문, 길 묻기 중 선택한 주제로 학생 프롬프트의 텍스트 치환
- 친구 앱 체험용 가독성·기능성·디자인 체크박스
- 로그인 지연 대응과 접속 불가 시 프롬프트 설계 활동(Plan B) 표시
- Tailwind utility 기반 반응형 레이아웃

다음 기능은 구현되어 있지 않습니다.

- Gemini 또는 다른 AI API 연결
- 프롬프트 전송과 AI 응답 수신
- 일본어 회화 문제 출제·정답 확인·채점
- 학생 결과물 업로드·공유·검토
- 체크리스트, 선택 주제 또는 진행 상태 저장
- 사용자 계정, 로그인, 역할과 학급 관리
- 데이터베이스, 서버와 분석 기능

페이지의 Gemini 로그인 안내와 퀴즈 생성 문구는 수업 절차 설명이며 이 저장소의 애플리케이션 기능이 아닙니다.

## 기술 구성

- HTML5
- 문서 내부 CSS와 JavaScript
- Tailwind CSS Play CDN
- Google Fonts의 Noto Sans KR
- 브라우저 Clipboard API

별도 패키지, 빌드 도구, 환경 변수와 JavaScript 모듈은 없습니다.

## 프로젝트 구조

```text
.
├── index.html  # 수업 콘텐츠, 스타일과 탭·복사 JavaScript
└── README.md
```

## 실행 방법

설치와 빌드 없이 `index.html`을 브라우저에서 열 수 있습니다. 다만 clipboard 동작은 브라우저의 secure context·권한 정책에 따라 `file://`에서 거부될 수 있으므로 로컬 개발 서버 사용을 권장합니다.

```bash
python -m http.server 8000
```

그다음 `http://localhost:8000/`에 접속합니다. Python은 편의를 위한 정적 서버 예시일 뿐 프로젝트 런타임 의존성은 아닙니다.

Tailwind CSS와 Noto Sans KR을 CDN에서 불러오므로 스타일을 온전히 표시하려면 네트워크 연결이 필요합니다. CDN이 차단되거나 실패하면 utility 스타일과 웹 폰트가 적용되지 않을 수 있습니다.

## 상호작용 한계

### 모바일 탭 접근

탭 버튼을 감싼 내비게이션은 Tailwind의 `hidden md:flex`를 사용합니다. 따라서 `md` breakpoint 미만의 모바일 화면에서는 내비게이션 전체가 숨겨지고 기본으로 열린 수업개요 외의 교사시연·학생실습·비상대응 탭에 접근할 대체 메뉴가 없습니다.

공개 전에 모바일 메뉴를 제공하고 탭을 button·tabpanel 의미 구조, 키보드 포커스, 선택 상태와 연결해야 합니다. 현재 탭은 URL hash나 브라우저 뒤로 가기 상태도 갱신하지 않습니다.

### 프롬프트 복사

복사 버튼은 `navigator.clipboard.writeText()`를 호출하고 성공 시 alert를 표시합니다. HTTPS 또는 localhost 같은 secure context와 clipboard 권한이 필요할 수 있습니다. promise 실패를 처리하는 `.catch()`, 실패 안내와 수동 선택 fallback이 없어 권한 거부나 미지원 브라우저에서는 사용자에게 원인이 표시되지 않습니다.

### 주제와 체크리스트

주제 카드는 학생 프롬프트 안의 표시 텍스트만 교체합니다. 선택 상태를 별도로 표시하거나 저장하지 않습니다. 체크박스도 현재 브라우저 화면의 일시적인 입력일 뿐 새로고침하면 사라지며 평가·제출·채점과 연결되지 않습니다.

### 콘텐츠 표시

수업개요의 핵심 메시지에 Markdown 강조 문법인 `**...**`가 일반 HTML 텍스트로 들어가 있어 별표가 그대로 화면에 표시됩니다. 강조하려면 `<strong>` 등 HTML 요소로 바꿔야 합니다.

footer의 `성취도 100%`와 `피드백 활발`, 본문의 `35분 완성`은 데이터로 계산되거나 실제 수업 결과에서 갱신되는 값이 아니라 고정 문구입니다. 근거가 없으면 목표·예시임을 표시하거나 제거해야 합니다.

## CDN과 브라우저 보안

- Tailwind Play CDN 스크립트는 버전이 고정되지 않았고 SRI가 없습니다.
- Google Fonts는 CSS `@import`로 외부 요청됩니다.
- Content Security Policy와 자체 호스팅 fallback이 없습니다.
- 외부 CDN 정책, 개인정보 영향, 학교 네트워크 차단 여부와 라이선스를 수업 전에 확인해야 합니다.
- Clipboard API는 secure context, 권한과 브라우저 지원을 실제 학생 기기에서 검증해야 합니다.

운영 공개 페이지라면 Tailwind를 고정된 로컬 CSS로 빌드하고 CSP·SRI·외부 리소스 최소화 방식을 검토하는 편이 안전합니다.

## 정적 배포

정적 파일이 저장소 루트에 있어 GitHub Pages의 branch 배포를 사용한다면 일반적으로 `main` 브랜치와 `/(root)` 폴더를 선택하는 형태입니다. 기존 README의 `/root`는 GitHub Pages 설정 UI의 표기와 다릅니다.

다만 저장소에는 Pages workflow, custom domain, 배포 URL 또는 실제 배포 성공을 증명하는 설정이 없습니다. 저장소 Settings의 외부 상태도 이번 작업에서 확인하지 않았으므로 현재 Pages 운영 여부를 주장할 수 없습니다. 배포 후에는 루트 URL, CDN 로드, 탭·clipboard, 모바일 화면과 HTTPS를 직접 검증해야 합니다.

## 수업 운영 주의사항

### Gemini와 학생 계정

이 페이지는 학생이 별도 Gemini 서비스에 로그인하고 프롬프트를 전송하는 활동을 전제로 합니다. 실제 수업 전에 학교 또는 교육기관 관리자가 다음을 확인해야 합니다.

- 학생 연령, 계정 유형과 보호자·기관 동의 요건
- 학교 관리형 계정에서 Gemini/Canvas 사용 가능 여부
- Google 및 조직의 최신 학생·교육용 AI 정책과 이용약관
- prompt와 생성 결과의 저장·학습 사용·보존·삭제 정책
- 학생 개인정보, 학급·학교 정보와 민감정보를 prompt에 입력하지 않는 지침
- 생성된 일본어 표현과 정답을 교사가 검토하는 절차
- AI 또는 네트워크를 사용할 수 없는 학생을 위한 동등한 대안

정책과 서비스 제공 범위는 변경될 수 있으므로 수업 시점의 공식 문서를 기준으로 다시 확인해야 합니다.

### 학급 정보와 교육 콘텐츠

페이지에는 특정 학과·반, 교사 시연 문구와 수업 성과처럼 보이는 표현이 포함됩니다. 공개 링크로 배포하기 전에 실제 학급 공개 승인, 개인정보·기관명 노출 범위와 학생 작업 공유 방식을 확인해야 합니다.

프롬프트, 일본어 교육 문구, 번역, 활동지와 디자인의 작성자·출처·재배포 권리는 저장소에서 확인되지 않습니다. 학교 수업이라는 목적만으로 외부 공개·수정·재배포 권리가 자동으로 생기지 않으므로, 교육 콘텐츠의 소유권과 제3자 자료 이용 조건을 수동으로 확인해야 합니다.

## 테스트 상태

자동화된 HTML·JavaScript 테스트, 접근성 검사, link 검사와 CI가 없습니다. 다음을 실제 수업 환경에서 확인해야 합니다.

- desktop 및 mobile 탭 접근과 키보드·스크린 리더 동작
- clipboard 성공·거부·미지원 상황의 안내
- 주제 치환 후 복사되는 프롬프트 내용
- 체크박스와 새로고침 동작
- 학교 네트워크에서 Tailwind·Google Fonts·Gemini 접근 가능 여부
- Gemini에 붙여 넣은 prompt의 결과 품질과 일본어 정확성
- Plan B 자료를 학생이 AI 없이 수행할 수 있는지
