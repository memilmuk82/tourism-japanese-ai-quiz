# AI로 만드는 일본 여행 회화 퀴즈 웹앱

관광일본어과 학생들을 대상으로 한 AI 융합 교류수업용 웹페이지입니다.

Gemini Canvas를 활용하여 일본 여행 상황별 회화 표현을 간단한 퀴즈 웹앱 형태로 만들어 보는 체험 수업 자료입니다.

## 수업 개요

- 대상: 관광일본어과 2반
- 주제: AI로 만드는 일본 여행 회화 퀴즈 웹앱
- 도구: Gemini Canvas
- 형태: 단일 HTML 정적 웹페이지
- 목적: 일본어 회화 콘텐츠를 AI를 활용해 디지털 결과물로 바꾸는 과정 체험

## 주요 내용

- 수업 개요 안내
- 교사 시연용 프롬프트
- 학생 실습용 프롬프트
- 일본 여행 상황별 회화 퀴즈 예시
- Gemini Canvas 활용 안내

## 실행 방법

이 프로젝트는 별도의 서버나 설치가 필요하지 않습니다.

브라우저에서 `index.html` 파일을 열면 바로 실행됩니다.

```text
index.html
````

## 배포 방법

GitHub Pages를 사용해 배포할 수 있습니다.

1. 저장소에 `index.html` 업로드
2. GitHub 저장소의 `Settings` 이동
3. `Pages` 메뉴 선택
4. Branch를 `main`, 폴더를 `/root`로 설정
5. 생성된 GitHub Pages 링크 공유

## 기술 구성

* HTML
* CSS
* JavaScript
* Tailwind CSS CDN

## 사용 목적

이 자료는 정규 평가용이 아니라 교류수업 및 AI 활용 체험용 자료입니다.

학생들이 코딩 문법을 깊게 학습하기보다는, 자신이 알고 있는 일본어 표현과 관광 상황 지식을 AI를 통해 웹페이지형 결과물로 바꾸는 경험을 하는 데 목적이 있습니다.
